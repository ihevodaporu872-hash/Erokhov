# Инженерный портал - Механические системы

Веб-приложение для управления и мониторинга механических систем здания: водоснабжения, вентиляции и отопления.

## Описание

Инженерный портал предоставляет современный веб-интерфейс для работы с основными инженерными системами:

- **Водоснабжение** - мониторинг давления, учет расхода воды, управление обслуживанием
- **Вентиляция** - управление режимами работы, контроль температуры и влажности
- **Отопление** - управление котельными, контроль параметров теплоносителя, настройка графиков

## Технологии

- **Node.js** - серверная платформа
- **Express** - веб-фреймворк
- **Bootstrap 5** - современный CSS-фреймворк для адаптивного дизайна
- **Bootstrap Icons** - набор иконок

## Установка и запуск

### Предварительные требования

Убедитесь, что на вашем компьютере установлен [Node.js](https://nodejs.org/) (версия 14 или выше).

### Шаги для запуска

1. **Перейдите в папку проекта:**
   ```bash
   cd engineering-portal
   ```

2. **Установите зависимости:**
   ```bash
   npm install
   ```

   Примечание: Если вы клонировали проект, этот шаг обязателен. Если вы только что создали проект по инструкции, Express уже установлен.

3. **Запустите сервер:**
   ```bash
   npm start
   ```

   Альтернативный способ:
   ```bash
   node server.js
   ```

4. **Откройте браузер:**

   Перейдите по адресу: [http://localhost:3000](http://localhost:3000)

5. **Остановка сервера:**

   Нажмите `Ctrl+C` в терминале

## Настройка порта

По умолчанию сервер запускается на порту 3000. Чтобы изменить порт, установите переменную окружения `PORT`:

**Windows (CMD):**
```cmd
set PORT=8080 && npm start
```

**Windows (PowerShell):**
```powershell
$env:PORT=8080; npm start
```

**Linux/macOS:**
```bash
PORT=8080 npm start
```
